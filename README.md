# 🌦 Vayumandal – Weather Data Analysis System

## 📌 Overview

**Vayumandal** is a Python-based weather data analysis system designed to explore historical climate patterns across multiple cities.
The project performs structured analysis of **temperature, humidity, wind speed, and air pressure** using real-world weather datasets.

This project demonstrates a complete data science workflow including **data cleaning, preprocessing, visualization, and statistical analysis** through a fully runnable Python script.

---

## 🚀 Features

• Independent and advanced analysis modes
• Monthly, yearly, and hourly trend analysis
• Multi-city comparative visualization
• Maximum, minimum, and average calculations
• Histogram and distribution insights
• CLI-based interactive workflow
